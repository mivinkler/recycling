Wareneingang erstellen

Hier legen Sie einen neuen Wareneingang mit Transportgut an.

Scanner-Hinweis

Über das Scanner-Symbol können Barcodes eingelesen werden. Akzeptiert werden Barcodes mit dem Präfix "G" (Generierter Barcode).

Tipps

  • Mindestens eine Zeile mit Material, Behälter und Gewicht erfassen.
  • Beim Gewicht sind Dezimaltrennzeichen erlaubt.
  • Leere Zeilen werden beim Speichern ignoriert.
  • Die Felder Lieferant und Lieferschein sind Pflichtfelder.
  • Für neue Wagen werden beim Speichern automatisch die Barcode mit der Prefix "L" (Lieferung) versehen.
{% elif selected_menu == "unload_create" %}

Vorsortierung erfassen

Hier erfassen Sie die Vorsortierung einer Liefereinheit. Das Transportgut aus dem Wareneingang wird in einzelne Wagen aufgeteilt und gewogen.

Oben können Sie neue Wagen anlegen, unten bestehende (aktive) Wagen aus früheren Lieferungen weiterverwenden.

Scanner-Hinweis

Über das Scanner-Symbol können Barcodes eingelesen werden. Akzeptiert werden Barcodes mit dem Präfix "L" (Lieferung).

Tipps

  • Wareneingang ist ein Pflichtfeld
  • Leere Felder sowie ganze Zeilen werden nicht gespeichert.
  • Über das Symbol    wird das Feld "Gewicht" automatisch ausgefüllt, alternativ kann das Gewicht manuell eingetragen werden.
  • Neue Wagen werden beim Speichern automatisch mit einem Barcode "S" versehen.
  • Nur ausgewählte Wagen (Checkbox) werden mit der Liefereinheit verknüpft.
