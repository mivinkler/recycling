Wareneingang erstellen

Wählen Sie einen Lieferanten aus und fügen Sie dessen Liefereinheiten (Transportgut) hinzu.

Hinweise

  • Über das aktivierte Scanner-Symbol    kann ein Barcode eingelesen werden. Akzeptiert werden Barcodes mit dem Präfix "G" (Generator-Code).
  • Beim Gewicht sind Dezimaltrennzeichen erlaubt.
  • Leere Zeilen werden beim Speichern ignoriert.
  • Das Feld "Lieferant" und mindestens eine Liefereinheit "Transportgut" sind Pflichtfelder.
  • Beim Speichern erhält jede Liefereinheit "Transpportgut" automatisch einen Barcode mit dem Präfix "L" (Lieferung)
{% elif selected_menu == "unload_create" %}

Vorsortierung erfassen

Hier erfassen Sie die Vorsortierung einer Liefereinheit. Das Transportgut aus dem Wareneingang wird in einzelne Wagen aufgeteilt und gewogen.

Oben können Sie neue Wagen anlegen, unten bestehende (aktive) Wagen aus früheren Lieferungen weiterverwenden.

Scanner-Hinweis

Über das Scanner-Symbol können Barcodes eingelesen werden. Akzeptiert werden Barcodes mit dem Präfix "L" (Lieferung).

Tipps

  • Wareneingang ist ein Pflichtfeld
  • Leere Felder sowie ganze Zeilen werden nicht gespeichert.
  • Über das Symbol    wird das Feld "Gewicht" automatisch ausgefüllt, alternativ kann das Gewicht manuell eingetragen werden.
  • Neue Wagen werden beim Speichern automatisch mit einem Barcode "S" versehen.
  • Nur ausgewählte Wagen (Checkbox) werden mit der Liefereinheit verknüpft.
{% elif selected_menu == "barcode_generator_create" %}

Barcode erstellen

Hier generieren Sie Barcodes, die das Ausfüllen der Formulare „Wareneingang erstellen“ und „Abholung erstellen“ vereinfachen. Stammkunden, die regelmäßig Lieferungen oder Abholungen durchführen, können feste Barcodes erhalten, um Formulare schneller auszufüllen.

Eingang-Barcode

Der Barcode wird im Formular „Wareneingang erstellen“ verwendet. Hier können Sie den Kunden festlegen und ein Transportgut mit den Werten Material, Behälter und Gewicht hinterlegen.

Abholung-Barcode

Der Barcode wird im Formular „Abholung erstellen“ verwendet.

Hinweis

Das Feld „Kunde“ ist ein Pflichtfeld. Leere Felder werden beim Speichern ignoriert.
