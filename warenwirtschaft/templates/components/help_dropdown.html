Wareneingang erstellen

Beschreibung

Hier wird neue Lieferung und die dazugehörigen Liefereinheiten (z. B. Container, Gitterboxen usw.) erfasst.

Liefereinheiten

  • Leere Zeilen werden beim Speichern ignoriert.
  • Beim Gewicht sind Dezimaltrennzeichen erlaubt.
  • Neu gespeicherte Liefereinheit erhält einen Barcode mit dem Präfix "L" (Lieferung). Dieser kann später in der Vorsortierung verwendet werden.
  • Eine Liefereinheit bleibt aktiv, bis sie in der "Vorsortierung" als eledigt markiert wird

Scanner

  • Über das aktivierte Scanner-Symbol kann ein Barcode eingelesen werden.
  • Akzeptiert werden Barcodes mit dem Präfix "G" (Generator-Code).

Vorsortierung erfassen

Beschreibung

Hier wird eine aktive Liefereinheit in kleinere Behälter (Wagen) umgefüllt und gewogen.

Wareneingang

  • Die Liefereinheit ist ein Pflichtfeld.
  • Wenn die gewählte Liefereinheit bereits verknüpfte aktive Wagen hat, werden diese automatisch angezeigt.
  • Bei einer Liefereinheit mit mindestens einem verknüpften Wagen (die Entladung läuft) wird die Schaltfläche Entladung abschließen angezeigt. Damit kann die Liefereinheit (z. B. Container) als Erledigt markiert werden.

Wagen

  • Um eine Zeile zu bearbeiten oder neu anzulegen, muss das Schloss-Symbol geöffnet sein .
  • Ein neuer Wagen erscheint nach dem Speichern in der Liste der aktiven Wagen.
  • Ein Wagen bleibt aktiv, bis er an die Abteilung Aufbereitung übergeben oder manuell auf Erledigt gesetzt wird.
  • Über das Symbol kann das Gewicht von der elektronischen Waage abgefragt werden. Das Feld Gewicht wird dabei automatisch ausgefüllt.
  • Neu gespeicherte Wagen erhalten automatisch einen Barcode mit dem Präfix S (Sortierung).

Scanner

  • Über das aktivierte Scanner-Symbol kann ein Barcode eingelesen werden.
  • Akzeptiert werden Barcodes mit dem Präfix L (Lieferung).

Barcode erstellen

Hier generieren Sie Barcodes, die das Ausfüllen der Formulare „Wareneingang erstellen“ und „Abholung erstellen“ vereinfachen. Stammkunden, die regelmäßig Lieferungen oder Abholungen durchführen, können feste Barcodes erhalten, um Formulare schneller auszufüllen.

Eingang-Barcode

Der Barcode wird im Formular „Wareneingang erstellen“ verwendet. Hier können Sie den Kunden festlegen und ein Transportgut mit den Werten Material, Behälter und Gewicht hinterlegen.

Abholung-Barcode

Der Barcode wird im Formular „Abholung erstellen“ verwendet.

Hinweis

Das Feld „Kunde“ ist ein Pflichtfeld. Leere Felder werden beim Speichern ignoriert.
