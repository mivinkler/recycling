Wareneingang erstellen

Beschreibung

Hier wird eine neue Lieferung und die dazugehörigen Liefereinheiten (z. B. Container, Gitterboxen usw.) erfasst.

Liefereinheiten

  • Leere Zeilen werden beim Speichern ignoriert.
  • Beim Gewicht sind Dezimaltrennzeichen erlaubt.
  • Neu Liefereinheit erhält bei der Speicherung einen Barcode mit dem Präfix L (Lieferung). Dieser kann später in der Vorsortierung verwendet werden.
  • Eine Liefereinheit bleibt aktiv, bis sie in der Abteilung Vorsortierung als eledigt markiert wird

Scanner

  • Über das aktivierte Scanner-Symbol kann ein Barcode eingelesen werden.
  • Akzeptiert werden Barcodes mit dem Präfix G (Generator-Code).

Vorsortierung erfassen

Beschreibung

Hier wird die eine aktive Liefereinheit in andere Behälter (Wagen) umgefüllt und gewogen.

Wareneingang

  • Wählen Sie zuerzst eine Liefereinheit aus (Pflichtfeld).
  • Wenn die gewählte Liefereinheit bereits verknüpfte (aktive) Wagen hat, werden diese automatisch angezeigt.
  • Bei der Liefereinheit mit mindestens einem zugeordneten Wagen (die Entladung läuft) wird die Schaltfläche Entladung abschließen angezeigt. Damit kann man die Liefereinheit (z.B. Container) als erledigt markieren.

Wagen

  • Ein Wagen bleibt aktiv, bis er an die Abteilung Aufbereitung übergeben oder manuell auf Erledigt gesetzt wird.
  • Über das Symbol kann das Gewicht direkt von der elektronischen Waage abgefragt werden. Das Feld Gewicht wird dabei automatisch ausgefüllt.
  • Neue Wagen erhalten beim Speichern automatisch einen Barcode mit dem Präfix S (Sortierung).

Scanner

  • Über das aktivierte Scanner-Symbol kann ein Barcode eingelesen werden.
  • Akzeptiert werden Barcodes mit dem Präfix L (Lieferung).

Status

  • Jede neu Liefereinheit erhält den Status Aktiv, bis sie in die Abteilung Vorsortierung übergeben wird.
{% elif selected_menu == "barcode_generator_create" %}

Barcode erstellen

Hier generieren Sie Barcodes, die das Ausfüllen der Formulare „Wareneingang erstellen“ und „Abholung erstellen“ vereinfachen. Stammkunden, die regelmäßig Lieferungen oder Abholungen durchführen, können feste Barcodes erhalten, um Formulare schneller auszufüllen.

Eingang-Barcode

Der Barcode wird im Formular „Wareneingang erstellen“ verwendet. Hier können Sie den Kunden festlegen und ein Transportgut mit den Werten Material, Behälter und Gewicht hinterlegen.

Abholung-Barcode

Der Barcode wird im Formular „Abholung erstellen“ verwendet.

Hinweis

Das Feld „Kunde“ ist ein Pflichtfeld. Leere Felder werden beim Speichern ignoriert.
